# Algorithm Interview Prep

This folder is for daily algorithm interview practice.

## Goal

Build steady problem-solving skill by practicing one theme per day instead of jumping between unrelated topics.

## How to use this folder

1. Open `PLAN_28_DAYS.md`
2. Practice the theme for today
3. Use `DAILY_TEMPLATE.md` to record what you learned
4. Revisit weak topics every 7 days

## Daily rhythm

- 15 min: review core pattern
- 45 min: solve 1 easy or medium warm-up
- 60 min: solve 1 medium main problem
- 30 min: rewrite solution without looking
- 15 min: summarize mistakes and pattern

Total: about 2 to 3 hours

If you are busy, keep the minimum version:

- 10 min review
- 35 min one problem
- 10 min recap

## Training principles

- Prefer depth over volume
- Write the idea before coding
- Track mistakes by pattern, not only by problem title
- Redo missed problems after 2 days, 7 days, and 14 days
- Spend more time on arrays, binary search, trees, DP, and graphs because they appear often in interviews

## Suggested problem sources

- LeetCode
- NeetCode topic lists
- Company-tagged interview sets after fundamentals are stable

## Files

- `PLAN_28_DAYS.md`: 4-week topic roadmap
- `DAILY_TEMPLATE.md`: daily reflection template
